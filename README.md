## How to Run

1. Clone the repository.

   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```

2. Open the Jupyter Notebook files.

3. Install the required Python libraries.

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

4. Run the notebooks in the following order:

   * Data Loading
   * Data Cleaning
   * Exploratory Data Analysis (EDA)

5. Execute the SQL scripts in PostgreSQL, MySQL, or SQL Server.

6. Open the Power BI dashboard (`.pbix`) to explore the interactive visualizations.

7. Review the report and presentation for a summary of findings and recommendations.
